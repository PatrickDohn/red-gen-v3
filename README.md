
# Resume Creator

Create, Design, and build your Resume.


## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform


## Run Locally

Clone the project

```bash
  git clone https://github.com/PatrickDohn/red-gen-v3.git
```

Go to the project directory

```bash
  cd app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** NextJs, React, React Hook Form, TailwindCSS, ShadCN, ReactPDFRenderer
